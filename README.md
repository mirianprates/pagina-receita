# Página de Receita Responsiva

Projeto simples desenvolvido com HTML5 e CSS3 para praticar estruturação de layout e responsividade.

## Sobre

Esta página apresenta uma receita organizada em:

- Imagem principal
- Título e descrição
- Lista de ingredientes dividida em duas categorias
- Modo de preparo
- Rodapé

O objetivo principal foi praticar:

- Estrutura semântica em HTML
- CSS Grid
- Media queries
- Organização de código
- Abordagem mobile-first

## Responsividade

- Em telas menores, o layout é em coluna única.
- Em telas maiores (≥ 768px), o conteúdo é organizado em duas colunas utilizando CSS Grid.
- A seção de ingredientes ocupa a largura total e é dividida internamente em duas colunas.

## Tecnologias

- HTML5
- CSS3
